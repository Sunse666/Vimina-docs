# Installation

## Download

Download the latest version from GitHub Releases:

```
https://github.com/Sunse666/Vimina/releases
```

## Run

Extract and run `Vimina.exe`, no installation required.

## First Launch

After launching, the program will display an icon in the system tray. Right-click the tray icon to:

- Show/Hide main window
- Open settings
- Exit program

## Verify Installation

1. Open any application window (e.g., Notepad)
2. Press ++alt+f++ to show control labels
3. If you see letter labels on the window, installation is successful
