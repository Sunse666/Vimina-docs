# 安装

## 下载

从 GitHub Releases 下载最新版本：

```
https://github.com/Sunse666/Vimina/releases
```

## 运行

解压后直接运行 `Vimina.exe`，无需安装。

## 首次启动

启动后，程序会在系统托盘显示图标。右键点击托盘图标可以：

- 显示/隐藏主窗口
- 打开设置
- 退出程序

## 验证安装

1. 打开任意应用程序窗口（如记事本）
2. 按 ++alt+f++ 显示控件标签
3. 如果看到窗口上出现字母标签，说明安装成功
