    内置函数¶

    鼠标函数¶

    click(x, y)¶

    左键点击指定坐标。

    1
    click(100, 200)

    clickR(x, y)¶

    右键点击指定坐标。

    1
    clickR(100, 200)

    dblclick(x, y)¶

    双击指定坐标。

    1
    dblclick(100, 200)

    move(x, y)¶

    移动鼠标到指定坐标。

    1
    move(500, 300)

    drag(fromX, fromY, toX, toY)¶

    从起点拖拽到终点。

    1
    drag(100, 100, 300, 300)

    scroll(delta)¶

    滚动鼠标滚轮。

    1
2
    scroll(1)    // 向上滚动
scroll(-1)   // 向下滚动

    getMousePos()¶

    获取当前鼠标位置。

    1
2
3
    var pos = getMousePos()
log(pos.x)   // X 坐标
log(pos.y)   // Y 坐标

    键盘函数¶

    type(text)¶

    输入文本字符串。

    1
    type("Hello World")

    key(keyName)¶

    发送按键或组合键。

    1
2
3
    key("enter")
key("ctrl+c")
key("alt+tab")

    keyDown(keyName)¶

    按下按键（不释放）。

    1
    keyDown("ctrl")

    keyUp(keyName)¶

    释放按键。

    1
    keyUp("ctrl")

    窗口函数¶

    activateWindow(title)¶

    激活指定标题的窗口。

    1
    activateWindow("记事本")

    closeWindow(title)¶

    关闭指定标题的窗口。

    1
    closeWindow("记事本")

    minimizeWindow(title)¶

    最小化指定标题的窗口。

    1
    minimizeWindow("记事本")

    maximizeWindow(title)¶

    最大化指定标题的窗口。

    1
    maximizeWindow("记事本")

    windowExists(title)¶

    检查窗口是否存在。

    1
2
3
    if windowExists("记事本")
    log("窗口存在")
end

    getWindowList()¶

    获取所有窗口列表。

    1
2
3
4
    var windows = getWindowList()
for win in windows
    log(win.title)
end

    控件函数¶

    clickTag(tag)¶

    点击指定标签的控件。

    1
    clickTag("AB")

    getControlByTag(tag)¶

    获取指定标签的控件信息。

    1
2
3
    var ctrl = getControlByTag("AB")
log(ctrl.name)
log(ctrl.bounds.x)

    getControlList()¶

    获取当前窗口的所有控件列表。

    1
2
3
4
    var controls = getControlList()
for ctrl in controls
    log(ctrl.tag + ": " + ctrl.name)
end

    时间函数¶

    sleep(ms)¶

    暂停执行指定毫秒数。

    1
    sleep(1000)  // 暂停 1 秒

    getTimestamp()¶

    获取当前时间戳（毫秒）。

    1
2
    var ts = getTimestamp()
log(ts)

    日志函数¶

    log(message)¶

    输出日志消息。

    1
2
    log("操作完成")
log("计数: " + count)

    截图函数¶

    screenshot()¶

    截取全屏。

    1
    screenshot()

    screenshot(filename)¶

    截取全屏并保存到文件。

    1
    screenshot("screen.png")

    screenshot(x, y, width, height)¶

    截取指定区域。

    1
    screenshot(100, 100, 400, 300)

    随机函数¶

    random(min, max)¶

    生成指定范围内的随机整数。

    1
2
    var r = random(1, 100)
log(r)

    randomFloat()¶

    生成 0-1 之间的随机浮点数。

    1
2
    var r = randomFloat()
log(r)

    数学函数¶

    abs(x)¶

    绝对值。

    1
    var a = abs(-5)  // 5

    floor(x)¶

    向下取整。

    1
    var f = floor(3.7)  // 3

    ceil(x)¶

    向上取整。

    1
    var c = ceil(3.2)  // 4

    round(x)¶

    四舍五入。

    1
    var r = round(3.5)  // 4

    sqrt(x)¶

    平方根。

    1
    var s = sqrt(16)  // 4

    pow(x, y)¶

    幂运算。

    1
    var p = pow(2, 3)  // 8

    系统函数¶

    beep()¶

    发出系统提示音。

    1
    beep()

    exit()¶

    终止脚本执行。

    1
2
3
    if error
    exit()
end

    getVersion()¶

    获取 Vimina 版本号。

    1
2
    var v = getVersion()
log(v)
